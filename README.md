# Eventify - Aplikasi Registrasi Event

Eventify adalah aplikasi web untuk pendaftaran event
yang memungkinkan peserta mendapatkan kode registrasi unik.

## Fitur
- Landing page event
- Form registrasi peserta
- Generate kode registrasi
- Login & dashboard admin
